## Requirements

* Python 2.7/Python 3.x
* Requests
* Requests[socks]
* Ruamel.yaml
* coloredlogs
* terminaltables
* raven (Sentry Client)

## Getting started

Edit **config.yml** with your credentials.
```bash
$ pip install -r requirements.txt

$ python main.py
```
